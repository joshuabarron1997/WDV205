What does a CSS preprocessor do?
A CSS preprocessor is used by web developers to process certain languages into css
What are the advantages of using a CSS preprocessor?
A preprocessor can do many various things: checking for syntax errors, use variables to call styles, the ability to call functions from outside sources, and possibly above all else, it can compile all of your styles into one sheet for uploading to the server.
What are the disadvantages or issues you need to consider when using a CSS Preprocessor?
You must change your already established work flow. Because all the sheets are compiles, line numbers are irrelevant for debugging which makes the process harder. You also must wait for the CSS to compile to see changes.
